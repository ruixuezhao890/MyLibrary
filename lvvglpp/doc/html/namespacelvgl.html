◆ CustomDeleter

template<auto DeleterFunction>
template< auto DeleterFunction > lvgl::CustomDeleter

Wraps a C deleter function for use with smart pointers.

This relies on the behaviour of the call operator that assumes that the left-hand side argument should be a function, and therefore gets cast to function pointer. At the same time, the integral_constant instance gives the wrapped value type as its own type.
